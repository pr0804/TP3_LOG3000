## Objectif
Ce projet est une calculatrice web simple développée avec Flask. Dans le cadre du TP, l'objectif est de mettre en pratique les bonnes pratiques de gestion de versions avec Git et GitHub. On va documenter le code fourni, créer des tests pour trouver les bogues, les corriger, et utiliser un workflow avec des branches et des pull requests pour collaborer en équipe.

## Prérequis d'installation
Avant de commencer, il faut s'assurer d'avoir installé sur le système :
Python 3.8 ou version supérieure
pip (gestionnaire de paquets Python)
Git
Un navigateur web moderne (Chrome, Firefox, Safari, Edge)

## Instructions d’installation et d’exécution
1. **Cloner le dépôt**
   git clone https://github.com/TP3_LOG3000.git 
   cd TP3_LOG3000
2. **Installer les dépendances**
   pip install flask
3. **Lancer l'application**
   python app.py
4. **Accéder à l’application**
   Ouvrir http://localhost:5000 dans le navigateur
